## 1. システム概要

銘柄コードを入力するだけで、財務データ、テクニカル指標、マクロ環境、および定性情報（有報/10-K）を統合的に解析し、プロフェッショナルな投資レポートと4軸スコアカードを自動生成します。

### 戦略一覧（v1.4時点）

| 戦略 | CLIオプション | 保有期間 | 設計思想 |
| :--- | :--- | :--- | :--- |
| **Long（中長期）** | `--strategy long` | 数ヶ月〜 | スコアベース。月次評価でATRトレーリング保有 |
| **Short（短期）** | `--strategy short` | 数日〜数週間 | スコアベース。日次ATRブラケット |
| **★Bounce（新規）** | `--strategy bounce` | 3〜7営業日 | 売られすぎからの逆張り反発買い |
| **★Breakout（新規）** | `--strategy breakout` | 5〜15営業日 | トレンド初動への順張りブレイクアウト |

## 2. BUY/SELLロジック（評価S版）

### 2.1 エントリー判定（共通・5段階）

| Step | 条件 | ブロック時 |
| :---: | :--- | :--- |
| 1 | 合算スコア ≥ BUY閾値（レジーム連動） | WATCH/SELL |
| 2 | 地力スコア ≥ 5.0 | BUYブロック |
| 3 | クールダウン経過（損切後15日 / 利確後7日） | 待機 |
| 4 | 株価 > MA75（Short/Bounce/Breakout限定） | BUYブロック |
| 5 | モメンタム確認（Short/Bounce/Breakout限定） | BUYブロック |

### 2.2 レジーム連動BUY閾値

| レジーム | BUY閾値 |
| :--- | :---: |
| RISK_ON | 5.5 |
| NEUTRAL | 6.5 |
| RATE_CUT | 6.0 |
| RATE_HIKE | 7.0 |

| YIELD_INVERSION | 8.0 |

### 2.3 Longエグジット（優先順位順）

1. **ATRトレーリングストップ**（高値からATR×3.0下落）

## 3. スイングロング戦略（v1.4 新規・詳細）

短期的な価格変動（数日〜15日程度）を捉え、資金効率を最大化するための戦略群です。

### 3.1 Pattern A: Bounce（反発買い）

売られすぎ水準からの自律反発を狙う「逆張り」戦略です。下落トレンドの一時的なリバウンド、またはレンジ相場での下限からの反発を拾います。

- **設計思想**: セリングクライマックス（出来高急増伴う投げ売り）直後の戻りをターゲットにする。
- **有効レジーム**: `RISK_ON`, `NEUTRAL`, `RATE_CUT`
- **保有期間**: 最大7営業日（短期決戦）

#### エントリー条件 (AND)
1. **RSI(9) < 32**: 明確な売られすぎ水準。
2. **株価 ≤ BB下限 (20, 2σ)**: ボリンジャーバンド-2σへの接触または突き抜け。
3. **出来高 ≥ 20日平均 × 1.3**: パニック売りの発生を示唆。
4. **地力スコア ≥ 5.0**: 反発する体力（ファンダメンタルズ）のある銘柄に限定。
5. **株価 > MA75**: 長期的な上昇トレンド中の押し目であることを確認。

#### エグジット条件 (優先順位順)
1. **ハードストップ**: `-2.5%`（常時発動）
2. **タイムストップ**: `7営業日` 経過かつ含み損（見込み違いとして即撤退）
3. **利確ライン**: `+5.0%`
4. **ATRトレーリング**: 含み益が `+2.0%` 経過後、高値から `ATR × 1.2` で追従。
5. **RSI反転**: `RSI(9) > 65` で利益確定。

---

### 3.2 Pattern B: Breakout（トレンド初動順張り）

保ち合いからの上放れ、または上昇トレンドの押し目からの再上昇を狙う「順張り」戦略です。

- **設計思想**: 強いモメンタムが発生した瞬間に乗り、トレンドが続く限り保有する。
- **有効レジーム**: `RISK_ON` のみ
- **保有期間**: 最大15営業日

#### エントリー条件 (AND)
1. **MA5 > MA25 (3日以内のGC)**: トレンド反転の初動を捉える。
2. **株価終値 > 直近20日高値**: レジスタンスの明確な突破。
3. **出来高 ≥ 20日平均 × 1.5**: "本物"のブレイクであることを出来高で裏付け。
4. **地力スコア ≥ 5.0**: ベースのある銘柄に限定。
5. **株価 > MA75**: 長期トレンドの追随を確認。

#### エグジット条件 (優先順位順)
1. **ハードストップ**: `-3.0%`
2. **タイムストップ**: `15営業日` 経過かつ含み損
3. **利確ライン**: `+8.0%`
4. **ATRトレーリング**: 含み益が `+3.0%` 経過後、高値から `ATR × 1.5` で追従。
5. **テクニカル手仕舞い**: `Dead Cross (MA5 < MA25)` で利益確定。

## 4. Long戦略改善案（v1.4追加・オプション）

### 4.1 Momentum Bonus（+0.5点）
完全MAアライメント（株価 > MA5 > MA25 > MA75）が確認できた場合、Technicalスコアに+0.5ボーナス。逆張り偏重の緩和。

### 4.2 優良割高株の特例BUY
地力スコア ≥ 8.0 かつ 合算スコア ≥ 5.5 の場合、通常の6.5閾値をバイパスしてBUYを許可。「良い企業を適正価格で買う」バフェット戦略に対応。

## 7. 残件・今後の課題 (Next Steps)

1. **シグナルの希少性への対応**: S評価レベルの厳格なフィルタにより、スイング戦略の取引回数が非常に少ない（20銘柄中4銘柄程度）。地力スコアの閾値をレジーム連動にするなどの緩和策の検討が必要。
2. **ウォークフォワード検証**: バックテスト結果の信頼性を高めるため、期間をずらしながら最適化する機能の実装。
3. **セクター別最適化**: セクター（自動車、半導体等）ごとにRSIやMAの期間設定を動的に変更するロジックの導入。
