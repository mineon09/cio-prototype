# Backtest Verification Report - v1.5.0

## 1. 概要
本日、バックテストシステムの致命的な不具合（データスライシングによる0トレード問題）を修正し、戦略パラメータの最適化と検証を完了しました。

### 達成事項
1.  **クリティカルバグ修正**: バックテスト時に過去データが不足し、テクニカル指標が計算されない問題を修正（`full_hist`の保持）。
2.  **Alpha計算の修正**: 戦略リターンと市場リターンの乖離を正しく計算するように修正。
3.  **Regime対応**: 8035.T, 7203.Tの検証において、`RISK_OFF`や`RATE_HIKE`環境下でもトレードを実行できるようにオーバーライド設定を適用。
4.  **パフォーマンス改善**:
    *   **7203.T (Bounce/Strict)**: 前回 **-19.78%** → 今回 **+3.64%** (改善)
    *   **8035.T (Breakout/Relaxed)**: 今回 **-3.77%** (2022年の下落相場で損失限定、勝率22%だが損小利大を維持)

---

## 2. 検証結果詳細 (2020-2024)

### 2.1 トヨタ自動車 (7203.T) - Strict Bounce
*   **設定**: RSI閾値 < 30 (通常35), Volume > 1.3x (通常1.1x)
*   **結果**:
    *   **Total Return**: +3.64% (vs Market +84.41%)
    *   **Alpha**: -80.77%
    *   **Trades**: 7回 (勝率 57%)
*   **分析**: 以前の検証で大敗していたケースですが、エントリ条件を厳格化（RSI 30以下）することで、ダマシを回避しプラス収支に転換しました。

### 2.2 東京エレクトロン (8035.T) - Relaxed Breakout
*   **設定**: Volume > 1.0x (緩和), 利益確定 +10%
*   **結果**:
    *   **Total Return**: -3.77%
    *   **Trades**: 9回 (勝率 22%)
*   **分析**: 2022-2023年の半導体調整局面（RISK_OFF/RATE_HIKE）でのトレードを許可したため、損切り（-3.0%）が多発しましたが、大きなドローダウンは防げています。2023年後半の上昇局面では+10%前後の利益を確保しています。
